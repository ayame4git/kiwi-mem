📝 人設編輯

修改後立即對新對話生效。支援模板變數：{user_name}、{assistant_name} 等。目前時間由網關自動注入對話，無需寫入人設（時間變數寫在這裡會破壞 Prompt 快取）。

🛠️ 工具

🌙 Dream 記憶整合

觸發 AI 的"睡眠"－整理碎片、合併記憶、產生前瞻。

📅 每日整理

從今天的聊天記錄產生日頁面，更新日曆和使用者畫像。

🧩 手動提取記憶

立即從最近對話中提取記憶碎片，不等自動輪次。

📦 導入預置記憶

从 seed_memories.py 導入預置記憶條目。

💾 資料備份

匯出所有對話、記憶、配置為 zip 文件。

🔄 向量遷移

為缺少向量的記憶補充 embedding（更換模型後使用）。
